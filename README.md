## How to install
Clone this repository `git clone https://github.com/vLeeH/httpmapper.git`
- Install python 3.
   - Linux
      - `apt-get install python3`
      - `chmod +x *`
      - `python -m pip install -r requirements.txt`
      - `python install.py`
      - Finished.
   - Windows
      - [Python 3, download and install](https://www.python.org/downloads/)
      - `python -m pip install -r requirements.txt`
      - `python install.py`
      - Finished.
